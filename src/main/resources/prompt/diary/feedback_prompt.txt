다음은 사용자가 작성한 학습 일지입니다.
GPT 너는 이 정보를 바탕으로 사용자의 학습 현황을 정리하고, 전문적인 피드백을 제공해야 합니다.
사용자는 매일 또는 주간 단위로 학습 목표를 설정하고 그에 따른 회고를 작성합니다.
아래는 해당 사용자가 오늘 작성한 실제 학습 기록이며, 총 9개의 정보를 포함하고 있습니다.

---

[1] 학습 제목
- 사용자가 오늘의 학습 일지에 입력한 제목입니다. 이 제목은 학습 주제의 핵심 키워드 또는 요약된 목표를 담고 있을 수 있습니다.
- 예: {title}

[2] 학습 내용
- 사용자가 오늘 수행한 학습 활동에 대한 자유서술형 기록입니다. 학습한 개념, 실습 내용, 느낀 점 등이 포함되어 있습니다.
- 예:
{content}

[3] 공부 시간
- 사용자가 직접 입력한 학습 소요 시간입니다. 단위는 분이며, 집중도나 학습량의 척도로 활용 가능합니다.
- 총 학습 시간: {studyTime}분

[4] 학습 만족도
- 사용자가 자신의 학습에 대해 평가한 주관적인 점수입니다. 0점에서 5점 사이의 숫자이며, 학습 경험의 감정적 만족도를 반영합니다.
- 만족도: {satisfaction} / 5

[5] 목표별 회고
- 사용자가 오늘 수행한 각 목표별로 작성한 회고문입니다. 회고는 목표별로 나뉘며, 각 회고는 짧고 간단할 수 있으나 핵심적인 성찰이 담겨있습니다.
- 예시 형식:
{goalReflection}

[6] 오늘 학습 중 어려웠던 점
- 사용자가 학습 중 겪은 어려움이나 막혔던 부분을 작성한 내용입니다. 기술적/인지적 문제점에 대해 GPT가 분석하고 조언할 수 있는 기초 자료입니다.
- 예: {difficulty}

[7] 내일 보완하고 싶은 점
- 사용자가 스스로 느낀 부족한 부분이나 내일 계획한 보완 전략입니다. 이 항목은 GPT가 다음 학습 방향을 제시할 때 참고해야 할 우선순위입니다.
- 예: {tomorrowPlan}

[8] 학습 목표 설정 배경
- 해당 목표를 설정할 때 사용자가 입력한 추가 설명입니다. 목표의 의미, 방향성, 학습 전략에 대한 의도가 담겨 있을 수 있습니다.
  - 목표 상세 설명: {goalDetail}
  - 목표 설정 이유: {goalReason}

[9] 사용자의 선호 학습 방식
- 사용자가 설정한 선호 학습 스타일입니다. 이 정보는 GPT가 피드백을 제안할 때 문체나 방식(이론/실습/문제풀이 중심)을 조정하는 데 활용할 수 있습니다.
- 선호 학습 방식: {learningStyle}

---

※ 위 9개 정보를 바탕으로, 다음 네 가지 항목을 반드시 순서대로 출력해주세요:

---

1. **오늘 학습에 대한 요약 (1~2문장)**
→ 핵심 개념/활동/결과를 간결하게 요약하세요.

2. **긍정적인 피드백 및 격려**
→ 사용자가 학습을 지속할 수 있도록 진심 어린 격려와 구체적인 칭찬을 제공하세요.

3. **문제점 및 학습 개선 방향**
→ 사용자가 작성한 ‘어려웠던 점’을 바탕으로, 학습 과정의 문제점과 구체적인 개선 방안을 제시하세요.

4. **다음 목표 또는 추천 학습 전략**
→ 사용자의 내일 계획, 목표 설명, 선호 학습 방식 등을 고려하여 실천 가능한 목표나 학습 전략을 추천하세요.

---

※ 반드시 **한국어**로 작성해주세요.
※ 각 항목 앞에는 **숫자를 붙여 번호 순서대로** 출력하세요.
※ 문장은 자연스럽고 따뜻한 말투를 사용하되, 실용적인 조언을 포함하세요.
※ 사용자의 학습 목적에 적합한 분석과 피드백을 제공해주세요.
